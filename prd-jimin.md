# PRD - 박지민

## 1. 문제 (Problem)
- 텔레그램 등 SNS 투자 리딩방에서 "무료 종목 리딩"을 미끼로 소액(약 10억 원 수준)만으로도 저유동성·소형주에 매수세를 몰아 급등시킨 뒤 빠지는 "묻지마 추종매수" 유도 패턴이 반복되고 있다.
- 리딩방 운영자가 5만 원 정도의 비용으로 소규모 인터넷 언론사에 보도자료성 기사를 뿌려, 세력이 만든 "숫자놀음"을 마치 진짜 기업 호재처럼 포장할 수 있다.
- 바이오 등 일부 업종은 뉴스 기사 하나에도 주가가 크게 흔들릴 만큼 개인투자자 피해에 취약하지만, 일반 투자자가 "이 추천이 진짜 호재인지 세력의 작전인지"를 스스로 판별할 근거 자료가 없다.
- 방향성 회의에서 (A) 사기 탐지형(리딩방 세력 자체를 잡아내는 것)과 (B) 건전 투자 유도형(좋은 기업으로 투자금이 흘러가도록 필터링하는 것) 중 **B로 확정**했다.

## 2. 목표 (Goal)
- "이상치 탐지 기법을 쓰는 것"이 아니라 **실제로 투자자를 더 잘 보호하는 도구**를 만드는 것이 최우선 목표다.
- 리딩방에서 추천된 종목의 뉴스·공시·재무 상태·거래량을 교차 검증해, "정상적인 호재 가능성"과 "묻지마 추종매수 위험"을 구분해 사전에 경고한다.
- 최종 판단은 왜 경고했는지 사람이 즉시 검증할 수 있는 **설명 가능한 규칙**이 내리고, 통계적 이상치 탐지(Isolation Forest)는 규칙이 놓칠 수 있는 패턴을 추가로 훑어보는 **보조 교차검증 신호**로만 사용한다(방법론이 목표에 종속되어야 하며, 그 반대가 되어서는 안 된다).

## 3. 사용자 (User)
- 리딩방·투자정보 채널의 추천을 접하는 개인 투자자, 특히 급등주·소형주·바이오주에 쉽게 휩쓸리는 초보 투자자.
- 리스크/준법 담당자 및 데이터 분석 담당자 등, FDS 관점에서 이상 종목·이상 보도를 모니터링해야 하는 내부 운영 인력.

## 4. 핵심 기능 (Features)
- **텔레그램 리딩방 크롤링**: Telethon 비동기 크롤러로 공개 채널 메시지를 수집(스노우볼 샘플링으로 채널 확장 포함).
- **추천 종목/기업 추출**: KRX 전체 상장 종목 사전(FinanceDataReader)과 매칭해 메시지에서 추천 종목명을 추출.
- **언론사 신뢰도 스코어링**: 한국언론진흥재단(KPF) 참여 언론사 명단 + 보완 매체 목록을 화이트리스트로 써서, 리딩방發 기사가 실린 매체의 신뢰도를 등급화("5만 원이면 기사를 뿌릴 수 있다"는 문제 대응).
- **뉴스 버스트·근접중복 탐지**: MinHash 기반 근접중복 탐지로 여러 매체가 짧은 기간에 거의 동일한 문구를 반복 게시하는 "작전세력 의심 보도" 패턴을 1차 판별.
- **뉴스-공시 상관관계 확인**: Open DART 공시검색 API로 추천 시점(전 7일·후 3일) 실제 공시 유무를 조회해, 뉴스만으로 만든 "숫자놀음"과 공시 기반 "진짜 호재"를 구분.
- **기업 건전성 스코어링**: Open DART(부채비율·감사의견·최대주주 지분변동·관리종목 이력)와 KRX 리스팅(시가총액·업종)으로 기업 안정성 및 "소액 개입에 취약한 소형주·바이오주" 취약도를 계산.
- **거래량 급등 탐지**: FinanceDataReader로 KRX 실제 일별 시세를 받아 20일 이동평균 대비 거래량 배율(300%+ 급등)을 계산.
- **설명 가능한 규칙 기반 최종 판정**: 위 신호들을 취약도 가중 동적 임계값 규칙으로 결합해 `투자 주의` / `정상 호재 가능성` / `특이사항 없음`으로 판정하고, 비교 종목이 5개 이상 쌓였을 때만 Isolation Forest로 통계적 이상치를 보조 교차검증.
- **결과 저장**: CSV/XLSX로 결과를 저장하고 Colab에서 바로 다운로드.
- 모든 외부 연동(텔레그램/네이버뉴스/DART)은 `DEMO_MODE` 토글로 API 키 없이도 내장 샘플 데이터로 전체 파이프라인을 테스트할 수 있다.

## 5. 성공 지표 (Metric)
- **북극성 지표 — 건전 투자 유도율**: 거래량이 급등(300%+)한 종목 중, 급등 이전에 "투자 주의" 경고가 이미 노출되어 있던 비율.
- **가드레일 지표 — 오탐률**: 삼성전자·SK하이닉스 같은 대형주가 여러 채널에서 동시 언급되거나, 정책 발표 후 테마 전체가 상승하거나, 실제 공시 직후 정상적으로 급등한 경우처럼 "반드시 확인할 반례"를 위험으로 잘못 판정하지 않는 비율.
- (보조 지표) 뉴스 버스트·근접중복 탐지의 정확도, 공시 매칭 성공률, 규칙 기반 판정과 실제 조사 사례의 일치율.
